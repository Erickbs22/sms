Atenção: Acesso Negado no Firebase!

O erro "Missing or insufficient permissions" que viu na consola significa que a aplicação conseguiu encontrar o seu projeto, mas o seu próprio Firebase está a bloquear a entrada por segurança.

Enquanto não corrigir as regras, a aplicação irá funcionar em Modo Local (os dados não vão para a nuvem). Para ativar a nuvem, siga estes passos:

  1. Aceda ao Console do Firebase.
  2. Vá a Firestore Database > aba Regras (Rules).
  3. Substitua o código existente por este e clique em Publicar:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
